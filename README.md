# Bluestock Mutual Fund Capstone Project - MJ28

## Project Overview
Analysis of Indian Mutual Fund industry from 2021-2025 using Python ETL pipeline and Power BI Dashboard. This project calculates risk metrics like VaR, CVaR, Sharpe Ratio and provides actionable investment insights.

## Key Features
- **ETL Pipeline**: Automated data ingestion and cleaning from 10+ raw CSV files
- **Risk Analytics**: VaR, CVaR, Sharpe Ratio, Correlation Analysis
- **Power BI Dashboard**: Interactive visualizations for fund performance
- **Recommendation Engine**: Cohort analysis and SIP continuity tracking

## Tech Stack
`Python` `Pandas` `SQLite` `Power BI` `Jupyter Notebook` `Git`
